# Frontend Assets

- `brand/logos/`: Koino wordmarks, symbols, and logo variants.
- `icons/`: App-specific icons that are not supplied by the icon library.
- `images/`: Content images, illustrations, and backgrounds.
- `fonts/`: Self-hosted font files.

Import assets from this directory through JavaScript or CSS so Vite can hash and
optimize them. Use `public/` only for files that must keep a fixed public URL.
